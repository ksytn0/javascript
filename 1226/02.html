내장객체_문자열(string)객채

자바스크립트의 내장객체 중에서 활용도가 많은 객체이며, 문자열 값을 인덱스번호를 통해 값을 찾거나 다른 문자로 병경도 가능하다.

응용 콘텐츠 - 아이디, 패스워드 입력창 문자열개수체크, 단문 메세지 문자열개수체크, 패스워드문자열 찾아서 다른 문자열로 변경, 이미지갤러리 등...

문자열 객체는 다른 내장 객체와는 다르게 new키워드를 생략해도 된다.

문자열 객체의 메서드와 속성들

  • replace - 문자치환하기
  • split - 문자열 분리하기
  • substring - 인덱스 번호 구간까지 문자데이터를 찾아줌
  • length - 문자열의 개수를 체크해줌
  • indexOf - 문자열과 일치하는 인덱스 번호를 반환

1. replace - 문자치환

2. split - 문자열 분리하기

3. substring - 인덱스 번호 구간까지 문자데이터를 찾아줌

4. length - 문자열의 개수를 체크해줌

5. indexof - 문자열과 일치하는 인덱스 번호를 반환
