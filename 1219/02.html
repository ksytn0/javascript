제어문02_switch case

선택문(switch case)은 사용자가 만든 조건이 만족할 때까지 계속 실행하면서 조건에 맞으면 실행되고 프로그램이 그자리에서 멈춤.

선택문은 조건식의 값이 정확히 일치해야 할 때 사용하면 이는 if문과의 차이점이다.

break문 - 조건식에서 원하는 값이 일치할 때 프로그램이 현 위치에서 멈추게 하기 위한 명령어.

        switch(변수){
            case 1 : 실행할 내용; break;
            case 2 : 실행할 내용; break;
            case 3 : 실행할 내용; break;
            default: 실행할 내용;
        }
