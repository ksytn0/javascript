데이터의 유형

자바스크립트에서 사용하는 데이터의 유횽에 대해 알아본다

  • 문자데이터형(string)
  • 숫자데이터형(number)
  • 부울형데이터(boolean) - true(확인), false(취소)
  • 찾을 수 없는 데이터형(undefined)
  • 값이 없는 데이터(null)
  • 배열형 데이터(array)
  • 함수형 데이터(function)
  • 객체(object) - document, window, alert, confirm, prompt, null ...

typeof - typeof연산자는 사용자가 입력한 데이터가 어떤 종류의 형식인지를 알려주는 연산자이며 변수앞에 Numer, String등을 작성하면 실제 데이터 형식을 알 수 있다.

es6문법에서 새롭게 추가된 기호 >> 템플릿리터럴(`) : 연산자를 사용하지 않아도 간단한 방법으로 새로운 문자열을 삽입할 수있는 기능을 제공한다, 이를 문자열 인터플레이션(String Interpolation)이라 한다.
