자바스크립트_연산자

자바스크립트 연산자는 사용자가 조건식을 만들어서 원하는 결과를 출력하고자 할때 사용하는 기호이다.

  • 산술연산자 : +, -, *, /, %(나머지
  • 문자열/결함연산자 : +(문자+문자, 문자+숫자 결함시 사용)
  • 비교연산자 : >, <, ==, ===
  • 대입연산자 : =, +=, -=
  • 증감연산자 : ++(1씩증가), --(1씩감소)
  • 논리연산자 : and, or, not
  • 삼항조건연산자 : 조건식? 조건식이 맍을 때 출력값은 '1', 조건이 맞지않을 때 출력값은 '2'
  • 타입연산자 : typeof(데이터타입을 구분하고 형식을 알려줌)
